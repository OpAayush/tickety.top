All Commands

help Get a list of commands and useful links.
setlogchannel Set a channel to be used as logchannel.
settranscriptchannel Set a channel to be used as transcript channel.
setup A way to easily set the bot up in 1 command.
supportrole Set a role to be used as general supportrole.
language Change the language tickety is displayed in.
add Add a user to your current ticket.
close Close the ticket you are currently in.
new Create a new ticket.
remove Remove a user from your current ticket.
rename Rename your current ticket.
ticketpurge Purge an amount of messages in a ticket.
transcript Create a free transcript of the ticket you are currently in.
claim Claim a ticket.
unclaim Unclaim a ticket.
blacklist Blacklist a user from creating tickets.
unblacklist Remove the blacklist from a user and allow them to create tickets again.
ticketembed Create a ticket open message in the channel your are currently in.
reset Reset tickety completely in your server.
botinfo Get the bot statistics.
dashboard Get a link to our dashboard.
invite Get an invite link to invite me to your Discord servers.
vote Get a vote link to vote on me.
serverinfo View basic information about this server.
sponsor Get a list of sponsors of Tickety.
support Get an invite to the support server of Tickety.
userinfo View your or someone else's information.
